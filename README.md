# About this py file
這是一個可以抓取自由時報生活版、國際版、地方版、政治版以及社會版的小程式。<br>
如果你會一點Python，那你可以在這個基礎上去抓取其他版。

## Function
這個小程式提供兩個Function：一個是使用關鍵字抓取過去新聞，另一個就是上面提到的。<br>
打開另一個功能的方式也很簡單，只要把程式碼第123行的「func_in = "2"」換成「input("Function --> 1:Keyword 2:Forums :")」即可。
